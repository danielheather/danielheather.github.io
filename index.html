Business Analytics Portfolio

Please find a selection of my portfolio below. The thumbnails link to the project descriptions and the titles link directly to their Github repositories.

PowerBI Report

A report into the sales, product and delivery performance of a mock European superstore. Offering insights into Customer Behavior, Shipping & Returns and Product Profitability.

K-Means Clustering

A showcase project of the use case of K-Means Clustering to interpret and categorize unstructured data.

Airline Overbooking Profitability Prediction Model

Made using Excel functions, this model is designed to maximise the profitabilty of an airline, by balancing the profit of overselling flight tickets with the cost of compsensating an overbooked passenger.

SQL

The utilization of SQL code to convert data into an easy-to-use database. This allows specific data queries without the risk of creating errors or issues in the dataset.

DAX

The utilization of DAX code to create new measures and columns. This allows you to better sculpt the visualisations.
